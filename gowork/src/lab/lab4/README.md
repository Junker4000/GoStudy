# GoWeb学习笔记
## 1. 使用 go get 下载安装 beego 与 bee 工具
```
$ go get github.com/astaxie/beego
$ go get github.com/beego/bee
```
## 2. 使用 bee 工具初始化 beego 项目
```
$ bee new myapp
```
## 3. 使用 bee 工具热编译 beego 项目
```
$ bee run myapp
```
## 4. 使用 go get 下载安装 beego ORM
```
$ go get github.com/astaxie/beego/orm
```
